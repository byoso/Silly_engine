# minuit introduction

minuit is a small terminal UI toolkit for prompts, forms, menus, and table-like displays.

## Included primitives

- TextField and print_formated for wrapped text output
- Field and ListField for validated user input
- Form for multi-field flows
- Menu and MenuItem for navigation
- Confirmation for yes/no prompts
- AutoArray for displaying list-of-dict data
- clear() for terminal clearing
